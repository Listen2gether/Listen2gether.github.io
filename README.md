Sync your listens on [ListenBrainz](https://listenbrainz.org) with other people.


Currently deployed at https://listen2gether.github.io/.

---

### How to compile

Requirements:
 - Dart Sass

Run the following commands:
```
nimble sass
nimble buildjs
nim r src/server.nim
```
